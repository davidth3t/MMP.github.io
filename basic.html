တာအတြက္ ပ႐ိုဂရမ္ ေတြေရးတဲ့အခါ သီးျခားအသံုးျပဳဖို႔ကန္႔သတ္ထားတာေတြရွိပါတယ္။ဒါေတြကို Keyword လို႔ေခၚပါတယ္။copy အတြက္ C,print အတြက္ P,Paste အတြက္ V,Move/Cut အတြက္ X ,Redoအတြက္ Y,Undo အတြက္ Z ဆိုတဲ့ Keyword ေတြကိုသတ္မွတ္ထားတဲ့အတိုင္းပဲသံုးၾကရပါတယ္။စံအျဖစ္ထားၿပီးအားလံုးကလိုက္နာက်င့္သံုးၾကတာလို႔ နားလည္ထားပါ။ဒီလို Keyword ေတြနဲ႔တူေနရင္ ေနာက္စာလံုးတစ္လံုး အစားထိုးေပးရပါတယ္။ဒါေၾကာင့္ File ေတြရွာတာမို႔ File explore,Windows စနစ္ႀကီးတစ္ခုလံုးအတြက္သံုးလို႔ Windows explore လို႔ေခၚတာကို Win+F, Win+W အဲလိုေတြမသတ္မွတ္ဘဲ Explore ရဲ႕ အစ အျဖစ္ Win+E လို႔သတ္မွတ္ထားတာျဖစ္ပါတယ္။ အသံုးမ်ားတဲ့တစ္ျခားဟာေတြကေတာ့ Desktop,Taskbar,Minimize (Maximize) တို့ပါပဲ။အစစာလံုးကိုမွတ္ဖို႔ျဖစ္လို႔ Desktop ဆို Win+D ႏွိပ္ရမယ္ဆိုတာသိသာမယ္ထင္ပါတယ္။Application တစ္ခုသံုးေနရင္းက Desktop ကိုၾကည့္ခ်င္တဲ့အခါ Win+D ကိုႏွိပ္ရပါမယ္။ဖြင့္လက္စ လုပ္လက္စ Application ကိုျပန္ဖြင့္ခ်င္ရင္ Win+D ကိုပဲ ျပန္ႏွိပ္ေပးရပါမယ္။ Toggle အေၾကာင္းေျပာခဲ့ျပီးပါၿပီ။ Taskbar ကို အသက္ဝင္ေအာင္ Active ျဖစ္ေအာင္လုပ္ဖို႔ Win+T ႏွိပ္ပါ။လိုခ်င္တဲ့ Application ေနရာေရာက္ေအာင္ Tab key ကိုလိုသလိုႏွိပ္ပါ။အဆံုးေနရာေရာက္ရင္ အစေနရာကိုျပန္ေရာက္ပါတယ္။ဒီလို ထပ္ခါတလဲလဲ ပတ္ေနတာကို ဒီမွာေတာ့ Circle လို႔သံုးပါတယ္။ေက်ာင္းမွာသင္ခဲ့ရတဲ့ Water Circle ေရသံသရာ ဆိုတာမ်ဳိးေပါ့။Tab key ကိုသံုးတဲ့အခါ လက္လြန္ျပီးေရွ႕ကိုေက်ာ္သြားတာမ်ိဳးရွိတတ္ပါတယ္။ေနာက္ကို ျပန္ဆုတ္ခ်င္တဲ့အခါ Shift key နဲ႔တြဲသံုးရပါမယ္။Tab key ႏွိပ္ရင္း ေက်ာ္သြားရင္ ေနာက္ကိုဆုတ္ဖို႔ Shift+Tab ႏွိပ္ရမယ္လို႔ဆိုလိုတာပါ။Shift ဆိုတာေျပာင္းလဲေပးတာပါပဲ။ ဖ​ြင့္ထားတဲ့ Application Window ေတြကို တျပိဳင္တည္းခ်ံဳ႕လိုတဲ့အခါ Minimize လုပ္ခ်င္ရင္ေတာ့ Win+M ေပါ့။ဒီေနရာမွာ Win+D နဲ႔ Win+M မတူ​ေၾကာင္းသိရပါမယ္။Win+D က လုပ္ေနလက္စ Active Window ကိုမခ်ံဳ႕ဘဲ Desktop ကိုျပဖို႔သံုးတာပါ။Win+D ကိုေနာက္တစ္ႀကိမ္ျပန္ႏွိပ္တာနဲ့ Active Window ကိုျပန္ျမင္ရပါမယ္။Win+M ကိုသံုးရင္ ဖြင့္ထားသမွ် Windows ေတြအားလံုးကို Minimize လုပ္သြားတာျဖစ္လို႔ ျပန္ေဖာ္ခ်င္ရင္ Shift+Win+M ႏွိပ္မွ Maximize ျဖစ္ပါမယ္။ (Active Windows) အသက္ဝင္ေနတဲ့ Window တစ္ခုကို ဘယ္ ညာ ခ်ဳံ႕ ခ်ဲ႕ စသည္ျဖင့္ ေနရာခ်ခ်င္တဲ့အခါ Win+Arrows keys ေတြသံုးႏိုင္ပါတယ္။ စက္တစ္ခုရဲ႕ အခ်က္အလက္ေတြကို ၾကည့္ခ်င္ရင္ Win+Pause/Break key ကိုႏွိပ္ပါ။ဒီမွာ စက္ရဲ့ အခ်က္အလက္ System Properties ေတြျမင္ရမွာပါ။ကိုယ့္စက္မွာ Windows Version ဘယ္ေလာက္တင္ထားသလဲ။ 32bit လား 64bit လား။RAM ဘယ္ေလာက္ရွိလဲ စသည္ျဖင့္ေပါ့။စက္ရဲ့ၾကံ့ခိုင္မွုအေျခအေနကိုသိခ်င္ရင္Rating မွာၾကည့္ပါ။ အဆင့္သတ္မွတ္ခ်က္ 1.0 to 9.9 ပါ။အမ်ားဆံုးက အေကာင္းဆံုးေပါ့။Rating နိမ့္ေပမယ့္ အေသးစိတ္မွာ Had
